# Meta6

Meta6 is a fan-made Bloons TD 6 strategy site for current metas, patch changes, hero tiers, paragon rankings, reusable strategies, map guides, and tower notes.

The site is built with React, TypeScript, and Vite. It uses source-linked BTD6 imagery from official press-kit material and community/wiki references where appropriate.

Live site: https://meta6-btd6-guide.vercel.app

## Features

- Dark, tabbed dashboard UI for fast navigation.
- Patch-aware meta cards and buff/nerf notes.
- Hero tier list with character portraits.
- Paragon tier list based on the Hbomb + MehalicPops pro ranking.
- Strategy library covering CHIMPS, bosses, heroes, paragons, freeplay, and map planning.
- Map-mode guidebook with 88 maps and 14 medal modes.
- Full tower roster with BTD6 tower images.
